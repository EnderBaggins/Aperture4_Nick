field_t
-------

.. doxygenclass:: Aperture::field_t
   :project: Aperture
   :members:

We define two typedefs to make it easier to use. A ``scalar_field`` is a ``field_t``
with only 1 component, and ``vector_field`` is a ``field_t`` with 3 components.
